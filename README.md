# Plano de Desenvolvimento Individual (PDI)

Projeto em Next.js para apresentar a jornada de desenvolvimento profissional, com foco em carreira, contribuições e evolução técnica/soft skills.

## O que é

Este site reúne meu PDI de estágio, organizando objetivos, entregas e aprendizados em páginas (abas) temáticas.

## Abas / Seções

- **Sobre**: visão geral do PDI e informações básicas.
- **Objetivo de Carreira**: trajetória profissional, foco por nível e metas em andamento.
- **Contribuições**: principais entregas e impactos no produto.
- **Roadmap**: stack de tecnologias por projeto e status de evolução.
- **Soft Skills**: habilidades comportamentais já praticadas e em desenvolvimento.

## Rodando localmente

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev`: inicia o ambiente de desenvolvimento
- `npm run build`: gera a build de produção
- `npm run start`: inicia a aplicação em produção
- `npm run lint`: executa o lint
